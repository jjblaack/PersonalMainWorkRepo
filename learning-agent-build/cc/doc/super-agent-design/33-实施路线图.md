## 风险与缓解

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| LLM API 成本高 | 高 | 中 | Token 预算、便宜模型降级 |
| 进程频繁崩溃 | 高 | 低 | 守护进程、状态持久化 |
| 记忆系统复杂 | 中 | 中 | 简化记忆类型、明确规则 |
| 工人管理复杂 | 中 | 中 | 从简单开始、逐步迭代 |
| 时间估算不足 | 中 | 高 | 预留缓冲时间、优先级调整 |
