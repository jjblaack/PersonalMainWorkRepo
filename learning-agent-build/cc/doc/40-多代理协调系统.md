## 1. 多代理系统概述

### 1.1 设计目标

多代理系统允许 Claude Code spawned 多个子代理并行工作，实现：

- **任务分解**: 将复杂任务拆分为子任务
- **并行执行**: 多个子代理同时工作
- **专业代理**: 不同代理专注于不同领域
- **结果聚合**: 汇总所有子代理的结果

### 1.2 架构层次

```
┌─────────────────────────────────────────────────────────────┐
│  主代理 (Main Agent)                                         │
│  - 接收用户请求                                              │
│  - 任务分解与分配                                            │
│  - 结果聚合与呈现                                            │
└─────────────────────────────────────────────────────────────┘
                            │
          ┌─────────────────┼─────────────────┐
          │                 │                 │
          ▼                 ▼                 ▼
┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐
│  子代理 A         │ │  子代理 B         │ │  子代理 C         │
│  (前端开发)       │ │  (后端开发)       │ │  (测试)          │
└──────────────────┘ └──────────────────┘ └──────────────────┘
```

### 2.3 执行流程

```typescript
// AgentTool 执行逻辑 (推断)

class AgentTool extends Tool {
  name = 'Agent';
  description = 'Spawn a sub-agent to work on a task';
  
  async execute(input: AgentToolInput, context: ToolContext) {
    // 1. 创建子代理会话
    const sessionId = generateSessionId();
    const childSession = await createSession({
      id: sessionId,
      parentSessionId: context.sessionId,
      initialPrompt: input.prompt,
      agentType: input.agent_type,
    });
    
    // 2. 启动子代理
    const result = await this.runAgent(childSession, {
      onProgress: (progress) => {
        // 向父代理报告进度
        context.onProgress?.({
          type: 'agent',
          sessionId,
          status: progress,
        });
      },
    });
    
    // 3. 返回摘要
    return {
      success: true,
      content: [{
        type: 'text',
        text: result.summary,
      }],
      data: {
        sessionId,
        fullOutput: result.fullOutput,
        tokenUsage: result.tokenUsage,
      },
    };
  }
  
  private async runAgent(session: Session, options: RunOptions) {
    // 启动子代理循环
    const engine = new QueryEngine({
      sessionId: session.id,
      model: session.model,
    });
    
    // 发送初始消息
    const response = await engine.send(session.messages, {
      onProgress: options.onProgress,
    });
    
    // 等待子代理完成
    while (!this.isTaskComplete(response)) {
      response = await engine.send(session.messages, {
        onProgress: options.onProgress,
      });
    }
    
    return {
      summary: this.extractSummary(response),
      fullOutput: response.content,
      tokenUsage: response.usage,
    };
  }
}
```

## 3. Coordinator 模块

### 3.1 协调器职责

**位置**: `src/coordinator/`

**核心职责**:
- 任务分解与分配
- 代理状态管理
- 结果聚合
- 冲突解决

### 3.2 协调器模式

```typescript
// Coordinator 类结构 (推断)

class Coordinator {
  private agents: Map<string, Agent>;
  private tasks: Map<string, Task>;
  private results: Map<string, AgentResult>;
  
  // 任务分解
  async decomposeTask(task: Task): Promise<SubTask[]> {
    // 调用 AI 进行任务分解
    const decomposition = await query([{
      role: 'user',
      content: `Decompose this task into subtasks:\n${task.description}`,
    }]);
    
    return parseSubtasks(decomposition);
  }
  
  // 分配任务
  async assignTasks(subtasks: SubTask[]): Promise<void> {
    for (const subtask of subtasks) {
      const agent = this.selectBestAgent(subtask);
      if (agent) {
        await agent.execute(subtask);
      } else {
        // 创建新代理
        const newAgent = await this.spawnAgent(subtask);
        this.agents.set(newAgent.id, newAgent);
      }
    }
  }
  
  // 聚合结果
  async aggregateResults(): Promise<FinalResult> {
    const results = Array.from(this.results.values());
    
    // 调用 AI 进行结果聚合
    const aggregation = await query([{
      role: 'user',
      content: `Aggregate these results:\n${results.map(r => r.output).join('\n---\n')}`,
    }]);
    
    return parseFinalResult(aggregation);
  }
  
  private selectBestAgent(task: SubTask): Agent | null {
    // 基于任务类型选择最合适的代理
    const candidates = Array.from(this.agents.values())
      .filter(a => a.canHandle(task));
    
    if (candidates.length === 0) return null;
    
    // 选择最空闲的代理
    return candidates.reduce((best, current) => 
      current.workload < best.workload ? current : best
    );
  }
}
```

```typescript
// TeamCreateTool (推断)

class TeamCreateTool extends Tool {
  name = 'TeamCreate';
  description = 'Create a team of agents for parallel work';
  
  async execute(input: { 
    name: string;
    agents: Array<{ type: string; specialty: string }>;
  }) {
    const team = {
      id: generateTeamId(),
      name: input.name,
      agents: [],
      createdAt: Date.now(),
    };
    
    // 创建团队成员
    for (const agentConfig of input.agents) {
      const agent = await createAgent({
        type: agentConfig.type,
        specialty: agentConfig.specialty,
        teamId: team.id,
      });
      team.agents.push(agent);
    }
    
    // 存储团队
    await saveTeam(team);
    
    return {
      success: true,
      content: [{
        type: 'text',
        text: `Created team "${input.name}" with ${team.agents.length} agents`,
      }],
      data: team,
    };
  }
}
```

## 5. 代理间通信

### 5.1 SendMessageTool

**位置**: `src/tools/SendMessageTool/SendMessageTool.ts`

**功能**: 代理间发送消息

```typescript
// SendMessageTool (推断)

class SendMessageTool extends Tool {
  name = 'SendMessage';
  description = 'Send a message to another agent';
  
  async execute(input: {
    recipientId: string;      // 接收者 ID
    message: string;          // 消息内容
    priority?: 'low' | 'normal' | 'high';
  }) {
    // 查找接收者
    const recipient = findAgent(input.recipientId);
    
    if (!recipient) {
      return {
        success: false,
        error: `Agent ${input.recipientId} not found`,
      };
    }
    
    // 发送消息
    await recipient.receiveMessage({
      from: context.agentId,
      content: input.message,
      priority: input.priority || 'normal',
      timestamp: Date.now(),
    });
    
    return {
      success: true,
      content: [{
        type: 'text',
        text: `Message sent to agent ${input.recipientId}`,
      }],
    };
  }
}
```

### 5.3 消息队列

```typescript
// 消息队列 (推断)

class AgentMessageQueue {
  private queues: Map<string, AgentMessage[]> = new Map();
  
  async enqueue(message: AgentMessage): Promise<void> {
    const queue = this.queues.get(message.to) || [];
    queue.push(message);
    this.queues.set(message.to, queue);
    
    // 通知接收者
    this.notifyRecipient(message.to);
  }
  
  async dequeue(agentId: string, limit = 10): Promise<AgentMessage[]> {
    const queue = this.queues.get(agentId) || [];
    const messages = queue.slice(0, limit);
    this.queues.set(agentId, queue.slice(limit));
    return messages;
  }
  
  private notifyRecipient(agentId: string): void {
    const agent = findAgent(agentId);
    if (agent) {
      agent.notifyNewMessage();
    }
  }
}
```

## 7. 性能考虑

### 7.1 并发限制

```typescript
// 最大并发代理数 (推断)

const MAX_CONCURRENT_AGENTS = (() => {
  const cpuCores = os.cpus().length;
  const memoryGB = os.totalmem() / (1024 * 1024 * 1024);
  
  // 基于 CPU 和内存动态调整
  return Math.min(cpuCores * 2, Math.floor(memoryGB), 10);
})();
```

### 7.2 资源管理

```typescript
// 代理资源管理 (推断)

class AgentResourceManager {
  private usedMemory = 0;
  private readonly maxMemory;
  
  async allocateForAgent(agentId: string, requiredMemory: number): Promise<boolean> {
    if (this.usedMemory + requiredMemory > this.maxMemory) {
      // 内存不足，需要等待或拒绝
      await this.waitForMemory(requiredMemory);
    }
    
    this.usedMemory += requiredMemory;
    return true;
  }
  
  releaseForAgent(agentId: string, memory: number): void {
    this.usedMemory -= memory;
  }
}
```

## 总结

多代理协调系统是 Claude Code 处理复杂任务的核心能力：

1. **任务分解**: AI 驱动的智能分解
2. **并行执行**: 多个代理同时工作
3. **专业分工**: 不同代理专注不同领域
4. **结果聚合**: 自动汇总所有结果
5. **代理通信**: 支持代理间协作

这个系统使得 Claude Code 能够处理远超单个代理能力范围的复杂项目。
